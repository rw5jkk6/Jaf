## 便利に使えるをwordlistをダウンロードする
- rockyou.txt.gz
  - `sudo gunzip /usr/share/wordlists/rockyou.txt.gz` 

- SecList
  - パスワードの辞書
  - `wget -c https://github.com/danielmiessler/SecLists/archive/master.zip -O /usr/share/wordlists/SecList.zip`
  - `cd /usr/share/wordlists`
  - `sudo unzip SecList.zip && sudo rm -f SecList.zip`

- sqlinjection
  - PayloadAllTheThingsを検索してSQL Injection / Intruder /SQL-Injectionをコピー
  - `sudo /usr/share/wordlists/`にsqlinjection.txtとして保存する 

- LocalFileInclusionのフォルダを作る
  - 検索する `gitee lfi list` 適当なサイトを開けばある。リストをコピーする
  - `cat > LFI-list.txt` ペーストする。抜ける
  - `sudo cp LFI-list.txt /usr/share/wordlists/`

- Hashcat rulesフォルダを作る
  - github hashcatを検索する、ブラウザでzipファイルをデスクトップにダウンロードする
  - デスクトップにあるzipファイルを解凍する
  - `cd ~/Desktop/hashcat-master`
  - `sudo cp rules/ /usr/share/wordlists/`
  - `sudo mv rules hashcat-rules`

### 辞書を環境変数に登録する
- 次のを`~/.bashrc`に書き込む
- 注意しておくことは、dirsmallはcommon.txtなので、例えばindex.htmlはあるが、dirbigなどはディレクトリしかないのでindex.htmlはない。必ずdirsmallをやってからdirbigなどを使う

```
#Dictionary of gobuster
export dirsmall=/usr/share/wordlists/dirb/common.txt
export dirmedium=/usr/share/wordlists/dirbuster/directory-list-2.3-small.txt
export dirbig=/usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt
```

## installシェルスクリプト
- Parrotでinstall.shを作って、コードをコピーして実行する。
- `chmod +x install.sh`
- `./install.sh`




```sh
#!/bin/bash

sudo apt update
sudo apt upgrade

echo 'start install'

sudo apt install -y \
 rlwrap \
 pipx \
 netcat-traditional \
 ftp \
 2to3 \
 knockd \
 steghide \
 expect \
 eom eog \
 freerdp2-x11 \
 ghex \
 hashid \
 exploitdb \
 cewl \
 wig \
 dhclient \
 hashid \
 hash-identifier \
 ltrace \

sudo apt remove samba-common samba-libs
sudo apt install -y smbclient

echo 'finish install'

sudo apt autoclean
```
